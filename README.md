# Fashion Hub UI Clone (Dark Aesthetic)

A modern, dark-aesthetic **Fashion Hub UI clone** built using **React** and **Tailwind CSS**.  
This project focuses on component-driven UI development, responsive layouts, and clean visual hierarchy while replicating a high-end fashion platform look.

---

## Overview

The Fashion Hub UI is designed to showcase independent designers, exclusive collections, and brand values using a bold dark theme.  
The entire UI was developed **from scratch**, breaking the layout into reusable React components and styling everything using Tailwind’s utility-first approach.

---

## Tech Stack

- **React (Vite)**
- **Tailwind CSS**
- **JavaScript (ES6+)**
- **Remix Icons**

---

## Project Structure

- `App.jsx` → Root layout & section composition  
- `Navbar.jsx` → Responsive navigation with reusable menu items  
- `Section1.jsx` → Hero section + feature boxes  
- `Section2.jsx` → Stats, about section & image layout  
- `Section3.jsx` → Advantages section with reusable rectangles  
- `Box.jsx`, `Rectangle.jsx`, `Square.jsx` → Reusable UI components  
- `DarkShade.jsx` → Overlay layer for hero background  
- `MenuItem.jsx` → Conditional UI rendering for CTA vs normal menu items  

---
